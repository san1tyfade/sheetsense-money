# Institutional Error Directory

This document serves as the master registry for error codes generated by Sheetsense. Use these codes when reporting issues to the code assistant.

## Authentication (ATH)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `ATH-100-01` | GSI Library Not Loaded | Check internet connection / Disable adblock |
| `ATH-401-01` | Token Expired | Silent re-auth trigger |
| `ATH-403-01` | Missing Scopes | Prompt user for full Drive.File permission |

## Google Infrastructure (GIO)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `GIO-404-01` | Spreadsheet Not Found | Prompt user to re-select via Picker |
| `GIO-429-01` | Quota Exceeded | Implement exponential backoff |
| `GIO-500-01` | Matrix Write Violation | Check if target sheet is protected/archive |

## Local Persistence (IDB)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `IDB-001-01` | Load Lock Violation | Prevented write before initial read complete |
| `IDB-500-01` | Version Mismatch | Force refresh / Migrated data check |
| `IDB-507-01` | Quota Exceeded | Prompt user to clear local archives |

## Intelligence Engine (INT)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `INT-500-01` | JSON Schema Mismatch | Fallback to deterministic regex parsing |
| `INT-401-01` | API Key Invalid | Verify Gemini API environment variables |

## Domain Logic (DMN)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `DMN-001-01` | Valuation Overflow | Check for zero-quantity holdings |
| `DMN-FX-01` | Forex Multiplier Missing | Fallback to USD=1.35 CAD |

## System Environment (SYS)
| Code | Meaning | Recovery Action |
| :--- | :--- | :--- |
| `SYS-500-01` | Uncaught Runtime Fault | System-wide crash; copy diagnostic payload |
| `SYS-100-01` | React Render Failure | Component failed to mount; check for undefined access |
